# Counter App

Una semplice applicazione web che simula un contatore. L'interfaccia è generata interamente tramite JavaScript con manipolazione del DOM.

## Funzionalità

- Valore iniziale impostato a **0**
- Pulsante **+** per incrementare il counter
- Pulsante **−** per decrementare il counter
- Pulsante **reset** per riportare il valore a 0

## Struttura del progetto

```
index.html        → shell HTML
css/style.css     → stili
js/counter.js     → logica (renderUI, updateCounter, displayValue, resetCounter)
```
